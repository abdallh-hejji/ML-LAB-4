# ML-LAB-4
Project Overview
The primary goal of this notebook is to transform the Chocolate_Sales.csv dataset into a clean, normalized format suitable for machine learning or advanced statistical modeling.

 Key Features & Tasks
Data Cleaning: Conversion of date strings to datetime objects and removal of currency symbols from financial columns.

Exploratory Data Analysis (EDA): Initial assessment of missing values and statistical distributions.

Imputation: Handling null values using median-based imputation.

Outlier Management: Detection and removal of anomalies using the Interquartile Range (IQR) method.

Feature Scaling: Application of both Min-Max Normalization and Z-score Standardization.

Dimensionality Reduction: Implementation of Principal Component Analysis (PCA) to evaluate variance.
